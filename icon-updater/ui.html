Imports icons into the current page. Duplicates are skipped automatically.

JSON format reference

Object keyed by category name (matches frame names on the current page, or auto-created). Each icon has a title and variants (30 variant keys → inner SVG markup).

{
  "AI & Magic": {
    "sparkle": {
      "title": "sparkle, star, magic, ai",
      "variants": {
        "filled=off, stroke=2, radius=0, join=square": "<path d=\"...\" stroke=\"currentColor\"/>",
        "filled=on, stroke=2, radius=0, join=square": "<path d=\"...\" fill=\"currentColor\"/>",
        "... 30 variants total ..."
      }
    }
  }
}
